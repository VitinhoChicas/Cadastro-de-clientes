# CadastroClientes

Sistema web para gerenciamento de cadastro de clientes com relatório de aniversariantes.

## Sobre o Projeto

O **CadastroClientes** é uma aplicação web desenvolvida em Java que permite o gerenciamento completo de informações de clientes, incluindo cadastro, edição, exclusão e geração de relatórios de aniversariantes por mês.

## Tecnologias Utilizadas

### Backend
- **Java 8**
- **Apache Tomcat**
- **Servlets/JSP**
- **PostgreSQL**

### Frontend
- **HTML5/CSS3**
- **JavaScript** - Interatividade

### Banco de dados

- **PostgreSQL**

## Funcionalidades

### Módulo de Clientes
- Listar todos os clientes
- Cadastrar novo cliente
- Editar informações do cliente
- Excluir cliente

### Módulo de Relatórios
- Relatório de aniversariantes do mês
- Filtro por mês específico ou todos os meses
- Opção de impressão do relatório
